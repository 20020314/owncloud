## Currently released:
========
### Roundcube
Embedd roundcube interface in owncloud

#### Requirements:
* OwnCloud 4.5+  (tested with 4.5.0 - 4.5.5)
* Mailserver with IMAP-Support
* Roundcube Webmailclient (tested with roundcubemail-0.8)
